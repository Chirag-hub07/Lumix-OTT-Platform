## 🚀 Overview

Lumix is a modern OTT streaming frontend built with React and Vite, delivering a fast, responsive, and visually engaging experience. It supports authentication, watchlist management, and API-driven content, replicating real-world streaming platforms with clean UI, scalable architecture, and smooth performance across devices.

---

## ✨ Features

* 🎥 Browse movies & shows
* 🔍 Smart search functionality
* ❤️ Watchlist management
* 🔐 Authentication system
* 🌐 API integration
* 📱 Fully responsive UI
* ⚡ Optimized performance
