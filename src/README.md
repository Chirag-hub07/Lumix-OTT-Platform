## 🚀 Overview

Lumix delivers a premium OTT browsing experience with a sleek interface, fast performance, and user-centric features. Built using modern frontend technologies, it combines authentication, API integration, and responsive design to simulate a real-world streaming platform with smooth navigation and engaging visuals.

---

## ✨ Features

* 🎥 Browse movies & shows
* 🧭 Category-based UI
* ⚡ Blazing fast (Vite)
* 📱 Fully responsive
* 🎨 Premium OTT-style design
